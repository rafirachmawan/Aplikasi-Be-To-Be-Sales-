# Mobile App

## Prasyarat

- Node.js LTS
- Yarn / npm / pnpm
- Expo CLI (jika pakai Expo)

## Setup

1. Salin `.env.example` menjadi `.env` lalu isi variabelnya.
2. (Opsional) Letakkan `google-services.json` (Android) dan `GoogleService-Info.plist` (iOS) di lokasi standar.

## Jalankan

```bash
# dari root
yarn      # atau npm i / pnpm i
yarn start
```
